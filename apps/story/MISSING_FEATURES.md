# Missing Features in Story Writing System

## Critical Gaps

### 1. Export Options
- **Current**: Only PDF export exists
- **Missing**:
  - DOCX/Word export
  - EPUB for e-readers  
  - Plain text/Markdown export
  - Full story JSON export (for backup/migration)

### 2. Collaboration Features
- **Current**: Has WebSocket infrastructure but no actual multi-user features
- **Missing**:
  - No sharing/permissions system
  - No real-time collaborative editing
  - No commenting/review system
  - No change tracking between users

### 3. Version History/Undo
- **Current**: Has backup service but no UI
- **Missing**:
  - No undo/redo functionality
  - No version history UI
  - No ability to revert to previous versions
  - No diff view between versions

### 4. Story Planning Tools
- **Current**: Basic chapter and character management
- **Missing**:
  - No outline/plot structure view
  - No story arc visualization
  - No timeline editor (only map timeline exists)
  - No relationship/connection graphs between characters
  - No world-building wiki/notes system

### 5. Writing Aids
- **Missing**:
  - No writing statistics/analytics dashboard
  - No consistency checker
  - No name generator
  - No writing prompts/inspiration tools
  - No style analysis

### 6. Advanced Search
- **Missing**:
  - Can't search across multiple stories
  - No advanced filters (by character, location, date range)
  - No regex search in UI

### 7. Organization Features
- **Missing**:
  - No story folders/categories
  - No tags system
  - No story templates/presets
  - No batch operations

### 8. Publishing Features
- **Missing**:
  - No cover image support
  - No metadata editor (ISBN, copyright, etc.)
  - No formatting templates
  - No direct publishing integrations

## Notes
The system has solid core functionality with AI integration, chapter management, character tracking, and map visualization, but lacks many features expected in a comprehensive story-writing platform.
